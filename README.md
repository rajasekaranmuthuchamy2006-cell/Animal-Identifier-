\# Animal Identifier 🐾



This project is an Animal Identification system built using Python and TensorFlow.

It predicts the animal from an input image using a trained CNN model.



\## Features

\- Image-based animal classification

\- CNN deep learning model

\- Supports multiple animal classes



\## Tech Stack

\- Python

\- TensorFlow / Keras

\- OpenCV

\- NumPy



\## Project Structure

\- train.py – model training

\- app.py – prediction app

\- animal\_model.keras – trained model

\- dataset/ – training images



\## How to Run

1\. Install Python

2\. Install dependencies  

&#x20;  pip install -r requirements.txt

3\. Run  

&#x20;  python app.py
